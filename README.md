# Todo CLI App

Консольное приложение для управления списком дел.

## Возможности
- Добавление задачи
- Удаление задачи
- Отметка задачи как выполненной
- Просмотр списка задач
- Сохранение данных между запусками в JSON

## Требования
- Python 3.9+

## Запуск приложения

```bash
cd app
python main.py
